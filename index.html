Guia de Intalação Para CNNs c/ Python (Windows):

Theano + Lasagne e TensorFlow + Keras


Theano + Lasagne

1. Instalar Conda

Conda é um gerenciador de pacotes, ambientes e dependências. Instale a versão para Python 3.6, que será ultilizada para instalar outras dependências.

Instale em C:\Users\Meu_Usuario\Anaconda3, e selecione a opção Add Anaconda to my PATH environment variable.

Faça download e instale o Conda.

2. Verifique a instalação

Execute os comandos:

conda -V

python -V

pip -V

Eles vão verificar se o conda, python e pip estão devidamente instalados e adicionados no PATH do Windows.

Execute depois o comando conda update -n base conda

3. Instalar o Theano e o Lasagne

Rode os comandos em sequência:

conda install theano

conda install -c toli lasagne

pip install --upgrade https://github.com/Lasagne/Lasagne/archive/master.zip

4. Configurar o Theano

Crie um arquivo .theanorc no diretório C:\Users\Meu_Usuario com o seguinte conteúdo:

          [global]
          floatx = float32
          mode = FAST_RUN
          device=cpu
          dnn.enabled=False
          [blas]
          ldflags = -LC:\Users\Meu_Usuario\Anaconda3\Library\bin -lmkl_rt

5. Testar funcionamento das bibliotecas e ferramentas

Faça download do script de teste do MNIST e execute o comando:

python mnist.py mlp 5



TensorFlow + Keras

1. Siga o tutorial do Theano + Lasagne

Ao seguir o outro tutorial, você já terá instalado o Conda, Python, Pip, e algumas dependências necessárias para utilizar o TensorFlow + Keras.

2. Criar um ambiente no conda para usar o TensorFlow

O Keras e o Lasagne utilizam versões diferentes do python, então criamos um ambiente com o conda e instalamos as dependências restantes.

Execute os comandos em ordem:

conda create -n tensorflow python=3.6

activate tensorflow

conda install -c conda-forge tensorflow

pip install --upgrade -I setuptools

conda install -c conda-forge keras

pip install tensorflow==1.2.0 --ignore-installed

3. Testar funcionamento das bibliotecas e ferramentas

Sempre que for usar o tensorflow + keras, execute os comandos:

activate tensorflow

set "KERAS_BACKEND=tensorflow"

Teste a biblioteca fazendo o download do script do MNIST e execute o comando:

python mnist.py

*Caso deseje usar outras bibliotecas, abra um novo prompt para sair do ambiente criado.

*Tutorial testado no Windows 10.
